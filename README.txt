MR.S Showcase V3

วิธีรัน:
1. เปิด terminal ในโฟลเดอร์โปรเจกต์นี้
2. รัน npm install
3. รัน npm run dev
4. เปิด http://localhost:3000

วิธีเปลี่ยนรูป:
- ใส่รูปใน public/images/backgrounds หรือ public/images/scripts
- รองรับ .png .jpg .jpeg .webp
- แก้ path ใน data/categories.ts และ data/scripts.ts ได้เลย
